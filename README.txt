Atividade: Desenvolver uma aplicação em java que navegue entre os dados (musica, documentos, imagens) de acordo com os comandos a baixo.  

• Comandos:
o SHOW mostra o conteúdo de um arquivo, não deve funcionar com diretórios
o LIST deve listar o conteúdo de um diretório
o BACK deve voltar um nível na árvore de diretórios, se já estiver na raiz, fica onde esta e mostra uma 
mensagem indicando que está na raiz
o OPEN deverá ser usado para abrir pastas
o DETAIL quando usado com arquivos ou diretórios, deve mostrar seus detalhes usando a 
BasicFileAttributeView do Java NIO 2
• Um leitor de arquivos deverá ser implementado para ler os arquivos, somente arquivos do tipo TXT devem 
ser suportados, quando um outro tipo de arquivo for informado como parâmetro, uma mensagem dizendo 
que “não há suporte ao arquivo” deve ser informada


Configurações:
-> Eclipse 
-> JDK 8
